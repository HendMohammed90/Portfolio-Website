# Portfolio Website

Personal portfolio for Hend Mohammed, built with Next.js (App Router), TypeScript, and Tailwind CSS.

## Requirements

- Node.js 20+ (recommended: latest LTS)
- npm 10+

## Install Dependencies

```bash
npm install
```

## Run the App

- Development:

```bash
npm run dev
```

Starts the app on `http://0.0.0.0:3000`.

- Production build:

```bash
npm run build
```

- Start production server:

```bash
npm run start
```

Starts the production app on `http://0.0.0.0:5000`.

- Lint:

```bash
npm run lint
```
